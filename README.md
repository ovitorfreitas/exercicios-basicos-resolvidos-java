## Exercícios básicos resolvidos (Lógica de programação em Java)

Questões:

1) Crie duas variáveis, uma para receber texto e outra para receber número. Atribua valores a elas e exiba os resultados na tela.

2) Crie uma variável para receber a nota de uma avaliação e outra para receber a nota de um trabalho. Atribua valores a elas e calcule a média. Após isso, exiba como saída na tela, a média e se o aluno foi aprovado (média a partir de sete) ou reprovado (média menor que sete).

3) Pergunte ao usuário seu nome, idade e cidade onde reside atualmente. Após isso, exiba na tela todos os dados que foram perguntados.

4) Pergunte ao usuário um número inteiro menor que cem. Após isso, caso o número digitado seja menor que cem, utilize uma estrutura de repetição para exibir na tela todos os números em sequência a partir do número que o usuário digitou até o número cem.

5) Pergunte ao usuário seu nome e se está empregado atualmente ou não. Enquanto o usuário cadastrado estiver empregado, faça um novo cadastro de usuário. O programa deve cadastrar usuários até que algum usuário cadastrado esteja desempregado. Ao identificar este usuário desempregado, deve retornar como saída, seu nome e uma mensagem dizendo que está desempregado. Ex: “João no momento está desempregado”. Após exibir a mensagem o programa deverá ser encerrado.
